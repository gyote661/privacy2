プライバシーポリシー

最終更新日：2026年4月30日

本プライバシーポリシーは、アプリ「禁煙チャレンジ」（以下「本アプリ」）における 利用者情報の取り扱いについて定めるものです。

1. 収集する情報

本アプリが収集・利用する情報は以下のとおりです。

  • 禁煙記録データ（成功・失敗の記録、連続日数、累計日数）
    端末内にのみ保存され、外部サーバーへは送信されません。
  • 広告配信に関するデータ
    本アプリはGoogle AdMob（Google LLC）による広告を表示します。 AdMobは広告の最適化・計測のため、デバイス識別子・IPアドレス・広告インタラクション等の情報を収集することがあります。

2. 情報の利用目的

  • 禁煙記録の表示・管理
  • 広告の配信および最適化

3. 第三者への情報提供

本アプリ自体が利用者の個人情報を第三者に販売・提供することはありません。 ただし、Google AdMobによる広告配信においては、Googleのプライバシーポリシーに基づいてデータが処理されます。

Google のプライバシーポリシー： https://policies.google.com/privacy

4. データの保存場所

禁煙記録データは利用者の端末内にのみ保存されます（AsyncStorage 使用）。 アプリ開発者のサーバーへのデータ送信は一切行っておりません。

5. 広告について

本アプリはGoogle AdMobを利用した広告を表示します。AdMobは パーソナライズ広告のためにCookieや類似の技術を使用する場合があります。 広告のパーソナライズを無効にしたい場合は、お使いの端末の広告設定からオプトアウトできます。

  • iOS：設定 → プライバシーとセキュリティ → 追跡 → トラッキングを許可をオフ
  • Android：設定 → Google → 広告 → 広告のパーソナライズをオプトアウト

6. 子どものプライバシー

本アプリは18歳未満のお子様を対象としていません。 18歳未満の方の個人情報を意図して収集することはありません。

7. プライバシーポリシーの変更

本ポリシーは必要に応じて更新されることがあります。 変更後のポリシーは本ページに掲載した時点で効力を生じます。
